# 🩺 The Ummah Clinic — عيادة الأمة

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **علل وأدوية (Ailments & Remedies)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 ailments + 20 remedies across 5 categories, blind driver interactive analogy, daily prescription, category-scored quiz, vital signs dashboard
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes with localStorage persistence
- Floating particles background, glassmorphism panels
- Swipe navigation, haptic feedback, reading mode
- Sound effects (Web Audio API)
- Responsive design + print styles
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
